Página de prueba (no usar con Samuel)

Esta página es solo para verificar, abriéndola en un navegador real, si estos sitios se pueden mostrar dentro de un iframe o si el navegador los bloquea (aparecerá una caja vacía, en blanco, o un mensaje de error si está bloqueado). No tiene el diseño ni la seguridad de SAMUEL OS — es únicamente una prueba técnica.

Cómo leer el resultado: si ves el contenido real del sitio dentro del recuadro, ese sitio permite ser embebido. Si el recuadro queda en blanco, o el navegador muestra un error tipo "se negó la conexión" / "refused to connect", ese sitio bloquea el embebido y no se puede usar dentro de SAMUEL OS.
